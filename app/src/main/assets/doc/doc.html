1. 如果您的"紧急联系人"未能成功收到短信，请务必按下列项目仔细检查程序的各项权限设置。
  2. 开启本程序后，如果您未接听“紧急联系人”的来电，程序会自动发送一条包含您当前位置信息的短信给“紧急联系人”。如：“上海市人民广场南京东路。经纬度：186.0000，168.0000”。
  3. 请仔细设置“紧急联系人”，以免泄露隐私和造成不必要的短信费用。
  4. 本程序需要您设置"定位"和"发送短信"权限为"允许"。注意：必须是"允许"，不能是"询问"。
  5. 为避免程序被异常结束，您需要把程序加入"白名单"。设置方法：
    • 华为：设置->权限管理->出行伞->设置单项权限->开启"应用自动启动"和"关联启动"选项
    • 小米：设置->授权管理->自启动管理
    • 魅族：设置->应用管理->出行伞->权限管理->后台管理->开启"允许后台运行"
    • 其他：参考以上设置
